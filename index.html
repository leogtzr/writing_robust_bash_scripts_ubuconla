Fail fast with

set -e

or:

set -o errexit

Example: ... 

						
#!/bin/bash
set -o errexit
cp "notfound" "x"
# something important with x file
# ...
exit
You can disable the behaviour with

set +o errexit

set +e

Or: 
						
cp "notfound" "hello.txt" || true

cp "notfound" "hello.txt" || :
						
					
						
#!/bin/bash
set -o errexit
# Critical code

# Disable error checking for
# non-critical code
set +o errexit
cp a b 
echo "Hello ... "

# When we are done ...
set -o errexit

echo "World"
exit 0
